# 超级英雄电影APP

## 此项目是根据vue最新版的脚手架和模板搭建而成
## 安装依赖包-如果在目录中没有node_modules文件夹，请运行以下指令
```
npm install
```

TIPS：如果项目运行错误，提示缺少依赖包等，可以删除`node_modules`文件夹，再重新运行以上指令安装一次依赖包

### 在开发环境中编译和启动

如果想在浏览器中看到代码效果，请运行以下指令：

```
npm run serve
```

### 在生产环境中编译并压缩代码（会变成普通代码，并出现个第dist文件夹）
```
npm run build
```

### 代码风格和错误检测
```
npm run lint
```
